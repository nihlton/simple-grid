# simpl-grid
simple responsive flexbox grid

DEMO: https://nihlton.github.io/simple-grid/

DOCS: https://nihlton.github.io/simple-grid/docs.html

# setup

In the console

``yarn add simpl-grid``

In your project

``import 'simpl-grid/grid.css';``
